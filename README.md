# Text Case Converter

A simple and professional web application to convert text into uppercase, lowercase, and title case.

## Features
- UPPERCASE conversion
- lowercase conversion
- Title Case conversion
- Real-time word and character count
- Copy to clipboard
- Clear text option

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Structure
text-case-converter/
│── index.html
│── style.css
│── script.js
│── README.md

## How to Use
1. Open index.html in a browser.
2. Enter text in the input box.
3. Click the required conversion button.
